Приветствую в документации по dxAdmin-cli

Здесь будут основная информация о моём плагином для сервера StarForce

Синтаксис - проще некуда !

На данный момент, в плагине есть несколько основных команды. Давайте посмотрим на их функционал, и синтаксис.

Все команды на момент написания.

Описание всех комманд

  • plyData - команда для получения информации о игроке или игроках
  • giveUniform - команда для выдачи звание, формы, оружия, игрокам
  • changename - команда для изменения имение на сервере
  • gotUniform - команда для получения новой формы с оружием

Посмотрим на таблицу всех команд а потом перейдём к примерам.

Назвние команды Первый флаг Второй флаг Третий флаг
plyData Номер игрока -a или --all, все игроки
giveUniform Легион Звание Номер игрока
changename Новое имя
gotUniform
